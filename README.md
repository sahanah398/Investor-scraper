# Investor Scraper Project

This repository contains a simple Python web scraping project for collecting investor/VC directory data.

## Included files
- `smergers_scraper.py` — working scraper for the Smergers investor directory.
- `requirements.txt` — Python dependencies for the scraper.
- `output/investors_smergers.csv` — scraped investor records generated by the script.
- `README_SMERGERS.md` — additional notes and usage details.

## What it does
The scraper:
- fetches the Smergers investor listing pages
- extracts investor name, type, website, stage, sectors, and profile link
- saves the data to a CSV file

## Run the scraper
Install dependencies and run:

```bash
pip install -r requirements.txt
python smergers_scraper.py
```

## Result
The script writes data to:
- `output/investors_smergers.csv`

## Notes
- The scraper uses `requests` + `BeautifulSoup`
- It handles pagination for the first 5 pages by default
- Some fields may require normalization after scraping
