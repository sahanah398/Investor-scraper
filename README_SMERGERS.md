# Investor Scraper — Smergers example

This repository contains simple Python scrapers that extract investor listings and save them to CSV.

Files
- `scraper.py` — starter/template scraper with configurable selectors (BASE_URL, selectors, PAGES).
- `smergers_scraper.py` — working scraper that collects investor cards from Smergers (pages 1–5 by default) and saves to `output/investors_smergers.csv`.
- `output/` — CSV outputs generated by the scripts.

Requirements
- Python 3.8+
- Install dependencies:

```bash
pip install -r requirement.txt
# or
pip install requests beautifulsoup4 pandas
```

Usage

Run the Smergers scraper (example):

```bash
python smergers_scraper.py
```

This will create `output/investors_smergers.csv` with columns:
- Investor Name
- Type
- Website
- Stage
- Sectors
- Profile Link

Customize and run the template scraper:
- Edit `scraper.py` and set `BASE_URL`, `PAGES`, and the CSS selector constants at the top of the file to match your target site.
- Run `python scraper.py` to produce `output/investors.csv`.

Notes and best practices
- Respect site `robots.txt` and the site's terms of service before scraping. Use rate limiting and polite headers.
- If pages are JavaScript-rendered, Requests + BeautifulSoup may not be sufficient; consider using Playwright or Selenium instead.
- The example scrapers use heuristics; some fields (stage, sectors, website) may be noisy and need normalization.
